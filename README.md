# 🚀 FastAPI Learning Journey

This repository contains my hands-on learning of FastAPI by building a simple Books API from scratch.

---

## 📚 What This Project Covers

- FastAPI setup and basics
- Uvicorn and ASGI understanding
- REST API design
- CRUD operations:
  - GET (fetch books)
  - POST (add book)
  - PUT (update book)
  - DELETE (remove book)
- Query parameters (filtering by author, category)
- Path parameters (fetch/update specific book)
- JSON file as temporary database
- Data validation using Pydantic
- Field Validator and Model Validator
- Path and Query Validation using Path/Query imported from fastapi .
